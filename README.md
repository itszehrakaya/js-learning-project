# Bulls and Cows Game on Web 

This is a simple web application built with HTML, CSS, and JavaScript, which allows the user to play the popular game Bulls and Cows. The game generates a random number with a user-selected number of digits and allows the user to guess the number. The game provides feedback to the user using + and - signs on how many digits they have guessed correctly and in the right position.

## Getting Started
To get started, simply open the numbergameweb.html file in your web browser. You can do this by either double-clicking on the file or by opening it through your web browser's "File" menu.

Or play on https://www.makerduragi.com/bulls-cows/

## How to Play
- The game generates a random number with a user-selected number of digits
- You enter your guess into the input field and click the "Guess" button.
- The game will provide feedback on how many digits you have guessed correctly and in the right position using + signs, and how many digits you have guessed correctly but in the wrong position using - signs.
- You can continue guessing until you guess the correct number.
- Once you guess the correct number, the game will display the number of guesses it took you to guess correctly.
